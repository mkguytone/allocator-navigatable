3.1. Linux カーネルのバージョンを揃える

本書では linux kernel の 4.18.16 を使って検証をしていますので、開発環境を揃えたほうが良いかと思います。

$ sudo add-apt-repository ppa:teejee2008/ppa
$ sudo apt-get update
$ sudo apt-get install ukuu

Ukuu のインストール後に「Ubuntu Kernel Update Utility」（または Ukuu Kernel Update Utility）を GUI から選択して以下の作業をしてください。

  1. 4.18 のインストール
  2. 4.18.16 のインストール

それとマルチブートが無効でインストールしてしまった場合は Ukuu の GUI メニューで有効にできた記憶があります。

  • Settings => Grub Options => Display GRUB menu during boot のチェックボックスを有効にする

「Display GRUB menu during boot」は「Set GRUB menu timeout」と表示される可能性もあります。

詳しくはオフィシャルサイトをチェックしてくださいね。

  • https://github.com/teejee2008/ukuu

まあ本を読むために絶対に必要とは言えないのですが、カーネルのバージョンを自分で変えられる状態で、本書を読んでると全く違う印象を持つかと思うので、重要なことです。
