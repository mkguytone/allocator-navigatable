71.2. カーネルソースのアーカイブ

　カーネルヘッダーのインストールだけでなく、ヘッダーとソースコードの両方が欲しい場合は、カーネルソースをホストするサイトからダウンロードすると良いでしょう。

https://www.kernel.org/
カーネルのアーカイブやマニュアルが入手できるサイトです。Linux Kernel Organizationが管理者なのでカーネルについての情報はこのサイトが信頼できるかと思います。

　本書で使用しているカーネル 4.18.16 のソースに限定するなら以下のようにダウンロードできる。

$ wget https://cdn.kernel.org/pub/linux/kernel/v4.x/linux-4.18.16.tar.xz
$ tar xf linux-4.18.16.tar.xz
$ cd linux-4.18.16/
$ ls
COPYING        Kconfig      README  crypto    include  lib      scripts   usr
CREDITS        LICENSES     arch    drivers   init     mm       security  virt
Documentation  MAINTAINERS  block   firmware  ipc      net      sound
Kbuild         Makefile     certs   fs        kernel   samples  tools

　wget を使わずともウェブサイトのホームページを見ながら自分の欲しいバージョンを選ぶことも可能なので、必ずしもコマンドラインに頼る必要はないです。
